# Changelog

All notable changes to the **Tuneora** project will be documented in this file.

## [1.0.0] - 2026-05-09
### Added
- **Initial Release**: Launching Tuneora as a premium Audio Player for Android.
- **Premium Design**: Implementation of the "Premium Airy" design system featuring glassmorphism and smooth micro-animations.
- **Advanced Media Engine**: Powered by Media3 (ExoPlayer) for high-fidelity audio playback.
- **Fast Delete**: Integrated support for bypassing system confirmation dialogs on Android 11+ via "All Files Access".
- **Volume Boost**: Loudness enhancement support up to 200% for studio-quality output.
- **Modern UI**: Full-width pill-shaped buttons and interactive preference sheets.
- **Smart Library**: Automated folder organization, universal search, and playback history.
- **Dynamic Themes**: Curated color accents and support for system dark/light modes.
